# 📄 Web currículo profissional

Este é o repositório do meu currículo online, desenvolvido com o framework Next.js. O projeto foi estruturado para apresentar minhas experiências profissionais e habilidades técnicas de forma moderna, rápida e responsiva.

---

## 🛠️ Tecnologias e Ferramentas

O projeto utiliza as tecnologias mais atuais do mercado para garantir performance e SEO:

* **Next.js:** Framework React para produção.
* **TypeScript:** Adição de tipagem estática para maior segurança no código.
* **Tailwind CSS:** Para estilização moderna e responsiva.
* **Geist Font:** Família de fontes otimizada da Vercel para melhor legibilidade.

---

## 🚀 Como Iniciar o Projeto

Para rodar este projeto localmente, siga os passos abaixo:

1.  **Instale as dependências:**
    ```bash
    npm install
    ```

2.  **Inicie o servidor de desenvolvimento:**
    ```bash
    npm run dev
    ```

3.  **Visualize no navegador:**
    Abra o endereço [http://localhost:3000](http://localhost:3000) no seu browser.

---

## 📁 Estrutura de Arquivos

* `app/page.tsx`: Onde reside o conteúdo principal do currículo.
* `next/font`: Sistema de otimização de fontes utilizado.

---

## 🌐 Deploy e Visualização

O projeto está configurado para ser hospedado na plataforma **Vercel**, que oferece integração nativa com o Next.js.
